AudienceNetwork

@objc(LIAudienceNetwork)
@objcMembers
public class AudienceNetwork : NSObject

Main interface to use Linkedin Audience Network in your app.

Start should be called as soon as possible and preferrably complete before Ad Requests are made in order to fill Bid Requests to Linkedin.

Warning

Call start(configuration:completion:) before accessing any other functionality to ensure initialization has occured.
  • bidderToken

    The Bidder Token to be passed to Bid Requests sent to Linkedin via the OpenRTB spec.

    This token is available if it was previously cached or upon successful completion of the start method. It will be internally cached and immediately available in subsequent calls.

    Warning

    If accessed before start this value will return an empty string.

    Warning

    Do not cache this value, instead, access it whenever needed.
